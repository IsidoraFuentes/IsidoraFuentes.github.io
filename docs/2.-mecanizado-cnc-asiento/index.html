Pasos para realizar el mecanizado

Paso 0
  • Crear plancha acorde al diámetro del asiento, 9mm de alto.
Paso 1
  • Posicionar la pieza horizontalmente al nivel de la plancha.
Paso 2
  • Cambiar el modo Design por el Manufacture y crear un Set Up con las siguientes características:

Adicionalmente, se debe configurar Stock -> From Solid -> seleccionar plancha.

Paso 3

Realizar un 2D Contour con las siguientes especificaciones:

  • Máquina: Flat end mill, 6 mm.

  • Seleccionar calados, línea inferior, para la geometría.

  • Passes -> Multiple Depths -> Max Rough Stepdown -> 1,4 aprox.

Paso 4

Realizar un Trace con las siguientes especificaciones:

  • Máquina: Flat end mill, 6 mm.

  • Sleccionar las 24 cadenas

  • Stalk to leave -> 4.5

Paso 6
  • Simular seleccionando botón derecho en el Set Up -> Simulate
