Silla Fresia

Pasos para realizar el mecanizado

Paso 0
  • Crear plancha 1220x1220, 15mm de alto.
Paso 1
  • Copiar y mover todas las piezas de la silla.
  • Separar las piezas y posicionarlas horizontalmente al nivel de la plancha.

Disclaimer: En este caso, el espacio no está siendo optimizado. Idealmente, se deben posicionar las piezas para que no queden partes vacías

Paso 2
  • Cambiar el modo Design por el Manufacture y crear un Set Up con las siguientes características:

Adicionalmente, se debe configurar Stock -> From Solid -> seleccionar plancha.

Paso 3

Realizar un 2D Contour con las siguientes especificaciones:

-Seleccionar máquina: Flat end mill, 6 mm.

-Seleccionar calados, línea inferior, para la geometría.

-Cambiar a Passes, luego Multiple Depths y en Max Rough Stepdown escribir 3mm.

Paso 4

Realizar un 2D Pocket con las siguientes especificaciones:

-Seleccionar máquina: Flat end mill, 6 mm.

- Seleccionar partes de menor milimetraje de las piezas.

-Cambiar a Passes, luego Multiple Depths y en Max Rough Stepdown escribir 3mm.

Paso 5

Realizar nuevamente un 2D Contour con las siguientes especificaciones:

-Seleccionar máquina: Flat end mill, 6 mm.

-Seleccionar calados, línea inferior, para la geometría.

-Cambiar a Passes, luego Multiple Depths y en Max Rough Stepdown escribir 3mm.

No olvidar: Tabs -> Width: 6mm -> Height: 3mm -> Distance: 350 mm.

Paso 6
  • Simular seleccionando botón derecho en el Set Up -> Simulate

¿Qué pasa con los calados interiores?

Cuando la fresadora corta por el interior del material, esta no lo hace en ángulo recto porque la fresa cala naturalmente en forma redonda y no tiene el espacio suficiente como para corregir este “error”. Es por eso que lo deja como esta señalado en amarillo. Sin embargo, a diferencia del caso anterior, al realizar los cortes exteriores, la máquina sí tiene el espacio suficiente como para simular un ángulo recto utilizando la misma cortadora, como podemos apreciar en color verde.
