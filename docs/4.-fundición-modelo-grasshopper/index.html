Pasos para realizar el mecanizado

Paso 1
  • Crear un sketch en el plano X en forma rectangular que rodee completamente el cuerpo

  • Extruir el sketch hasta cubrir el objeto en su totalidad



Paso 2
  • Cambiar settings a Manufacture
  • Crear un nuevo Set Up con las siguientes especificaciones:

Adicionalmente:

  • Configurar Stock -> From Solid -> seleccionar plancha
Paso 3

Editar la herramienta de fresado:

  • Abrir Edit Tool
  • Duplicar la máquina clásica que usamos: Flat end mill, 6 mm.
  • Editar nombre y propiedades como se resalta en las siguientes imágenes:



Paso 4

Cambiar settings a Design nuevamente:

  • Seleccionar create sketch -> tecla “P” -> plano X -> seleccionar cuerpo -> OK
    Al hacer esto, se crea un rectángulo que simula el diámetro exacto de la figura.

Paso 5
  • Volver al setting de Manufacture

Seleccionar Adaptative Claring:

  • Máquina: la que creamos anteriormente (duplicado flat end mill 6mm)
  • Configurar Passes -> Optimal Load -> 3mm
  • En Geometry seleccionar las opciones que muestra la imagen:



Al seleccionar OK, se va a demorar un par de minutos en cargar, ya que la fresadora tiene que generar muchos caminos designados a la mecanización adaptativa:

Al finalizar, el recorrido se verá así:

Paso 6

Seleccionar Parallel

  • Máquina: Cambiamos a ball end mill, 6mm.

  • Configurar Passes -> Stepover -> 3mm

  • En Geometry seleccionar las opciones que muestra la imagen:

Al seleccionar Ok, de debería mostrar algo así:

Paso 7

  • Duplicar setup Parallel

  • Editar passes -> pass direction -> 90°

  • Editar passes -> stepover -> 2 mm

Al unir el paso 6 con el 7, se debería mostrar un recorrido como el siguiente:
